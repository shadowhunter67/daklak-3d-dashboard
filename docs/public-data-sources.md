`DOCUMENT_REFERENCES` (`src/data-platform/catalog/documents.ts`). "Integration status" follows the
spec's five values: `integrated` (real data, actually rendered today), `adapter-ready` (a real
adapter exists but no live dataset uses it yet), `document-only` (a text citation, no spatial
data), `research-needed` (found but not independently verified), `not-permitted` (considered and
rejected).

## Sources considered and not pursued this pass

- **Bulk population/economy statistics** beyond the one province-overview dataset above — spec §9
  explicitly forbids bulk scraping in this pass; would need a per-indicator provenance review, not
  a bulk import.
- **Quy hoạch sử dụng đất / danh mục dự án kêu gọi đầu tư** (spec §2.3) — real portals exist
  (`khdt.daklak.gov.vn`, `sotaichinh.daklak.gov.vn`) but this pass only verified one document
  citation; a real dataset here needs someone to review each project/zone entry's own provenance
  individually, not just cite the portal.
- **GSO / Tổng cục Thống kê provincial spatial statistics system** — named in spec §2.2 as a
  priority source; not investigated this pass (research-needed, not attempted).
